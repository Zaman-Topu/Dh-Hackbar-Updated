## 🚀 Overview
**DroidHack Pro** is the revived and updated version of the legendary DH-HackBar penetration testing tool, fully optimized for modern Android devices (Target SDK 34). It brings powerful web security assessment capabilities directly to your smartphone.

Whether you are performing SQL Injections, Cross-Site Scripting (XSS) assessments, or bypassing Web Application Firewalls (WAF), HackBar provides an integrated toolkit right inside a specialized browser.

## ⚡ Key Features
- **Comprehensive Payloads:** Pre-loaded with hundreds of vectors for SQLi, XSS, LFI, RCE, and more.
- **Admin Panel Scanner:** Multi-threaded fast scanner to uncover hidden admin directories.
- **URL Balancer & Tampering:** Modify headers, post data, and manipulate requests on the fly.
- **Custom Queries:** Add, save, and manage your own custom exploit payloads.
- **Modernized Backend:** Re-written for Android 12+ compatibility with `ExecutorService` and `FileProvider`, preventing legacy crashes while keeping the classic UI intact.
- **Automated CI/CD:** Fully integrated with GitHub Actions for automated, reliable APK builds on every push.

## 🛠️ Built-in Tools
* **SQLi:** Union, Error-Based, Double Query, DIOS, MSSQL, PostgreSQL
* **Exploitation:** LFI, RCE, XPath Injection
* **Bypass:** WAF Bypass, Order By Bypass, Auth Bypass, User Privileges
* **Utilities:** String manipulation, Base64 encode/decode, MD5 hashing, Extractor

## 📥 Installation
1. Go to the [Releases](https://github.com/Zaman-Topu/Dh-Hackbar-Updated/releases) page.
2. Download the latest `app-release-unsigned.apk` or `app-debug.apk`.
3. Install the APK on your Android device. *(You may need to allow installation from unknown sources).*
